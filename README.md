# sparql-queries
grlc compatible collection of SPARQL queries for multimodalities.

Overview of all available SPARQL queries for this repo: https://grlc.io/api-git/MuseIT-project/sparql-queries/

Example of grlc request to get overview of keywords stored in MuseIT experimental knowledge graph: 
https://grlc.io/api-git/MuseIT-project/sparql-queries/KeywordsOverview?endpoint=https%3A%2F%2Ftriplestore.coronawhy.org%2Fnowmuseum%2Fsparql
